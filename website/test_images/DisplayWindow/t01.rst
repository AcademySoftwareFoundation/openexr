The display window and the data window are the same.  All pixels
in the data window should be visible in the display window.

.. list-table::
   :align: left

   * - screenWindowWidth
     - 1
   * - dataWindow
     - (0 0) - (399 299)
   * - screenWindowCenter
     - (0 0)
   * - displayWindow
     - (0 0) - (399 299)
   * - channels
     - B, G, R
   * - lineOrder
     - increasing y
   * - pixelAspectRatio
     - 1
   * - compression
     - piz
   * - type
     - scanlineimage
