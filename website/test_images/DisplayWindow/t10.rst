The display window and the data window do not overlap.  The
entire display window should be filled with the background
color.

.. list-table::
   :align: left

   * - screenWindowWidth
     - 1
   * - dataWindow
     - (0 0) - (399 299)
   * - screenWindowCenter
     - (0 0)
   * - displayWindow
     - (-100 0) - (-1 299)
   * - channels
     - B, G, R
   * - lineOrder
     - increasing y
   * - pixelAspectRatio
     - 1
   * - compression
     - piz
   * - type
     - scanlineimage
